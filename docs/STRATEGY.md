## 产品定位

### Anima Agent（底座·免费）
基于 OpenClaw MIT 许可深度定制的中文优先 AI 智能体底座。内置灵元三件套：
- 🆔 **MeshIdentity** — 去中心化身份锚定
- 🛡️ **MemGuard** — 记忆安全加密保护
- 🧭 **Polaris** — 人格基线防漂移

**永远免费** — 是灵元生态的流量入口和品牌抓手。

### LingOS（人格操作系统·付费）
搭载蒸馏 AI 人格的操作系统。一个 LingOS = 一个特定角色的 AI Agent。
- **LingOS CE** — 免费社区版，通用助手
- **Nyx v1 / Kronos v1 / Shun v1** — ¥199 买断，蒸馏人格
- **LingOS Pro** — ¥499，多人格切换
- **LingOS Team** — ¥1,999/年，企业团队版
- **LingOS Enterprise** — ¥9,999起/年，私有部署+定制

## 商业模式

```
Anima Agent（免费获客）
      │
      ├──→ LingOS 付费版（一次性买断）
      ├──→ Token 分成（与模型厂商持续分成）
      └──→ 企业版（年付高客单价）
```

## 飞轮效应

更多免费用户 → 更大 Token 议价权 → 更多分成收入 → 升级付费版 → 口碑传播 → 更多用户

## 差异化壁垒

别人 Fork OpenClaw → 得到一个 Agent 框架。
我们 Fork OpenClaw → 每个用户在加入一个有身份的网络。

"框架" vs "网络" — 这是本质区别。
